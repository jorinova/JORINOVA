🔐 Set Up 2FA for My Account

Two-Factor Authentication adds a second layer of security using TOTP (Time-based One-Time Password, RFC 6238). Compatible with Google Authenticator, Authy, Microsoft Authenticator.

ℹ️ How 2FA Works in ALIS-X
1️⃣ Enable 2FA — Generate a secret key and scan the QR code with your authenticator app
2️⃣ Verify — Enter a 6-digit OTP from your app to confirm setup
3️⃣ Login — After password, enter the current OTP from your app
4️⃣ Codes refresh every 30 seconds — Use the current code shown in your app
⚠️ Store your backup secret key securely. If you lose your device, you will need it to restore access.
